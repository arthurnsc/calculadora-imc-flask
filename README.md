# Calculadora de IMC com Flask 

Esta é uma aplicação web simples, desenvolvida em Python com Flask, HTML5 e CSS3, que permite calcular o Índice de Massa Corporal (IMC).

## Funcionalidades 

- Cálculo automático do IMC a partir do peso e altura.
- Classificação do resultado.
- Layout responsivo.
  
## Como usar

Clone o repositório, instale o Flask, execute o arquivo main.py e acesse http://127.0.0.1:5000/
